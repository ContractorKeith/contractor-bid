# Code of Conduct

## Our Standard

This project is for commercial construction people, estimators, developers, and AI-tool builders who want better bid workflows. Contributors are expected to keep discussion practical, respectful, and focused on improving the work.

Examples of welcome behavior:

- Use clear, specific feedback.
- Assume good intent, but correct bad information directly.
- Respect that trade knowledge, field experience, and software experience all matter here.
- Keep proprietary bid documents, customer files, and private project details out of public issues and pull requests.
- Disagree on technical decisions without making it personal.

Examples of unacceptable behavior:

- Harassment, threats, insults, or targeted personal attacks.
- Publishing private information or proprietary bid material without permission.
- Sexist, racist, or otherwise discriminatory language.
- Spam, trolling, or repeated off-topic arguments.
- Pressuring maintainers or contributors to provide unpaid estimating, pricing, or legal advice.

## Scope

This Code of Conduct applies in project spaces, including GitHub issues, pull requests, discussions, documentation, and any community channel created for this project.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block participants when behavior harms the project or community. Serious or repeated violations may result in a permanent ban from project spaces.

If you need to report a problem, contact the maintainers through the repository owner account. Include relevant links, screenshots, or context when possible.

## Safety Note

This project helps organize bid information. It does not replace professional judgment, contract review, local code review, or trade-specific estimating responsibility.
